Listing attributes and relationships

BaseTen Assistant can list entitites' attributes and relationships like they are visible to BaseTen.

For relationships to be listed correctly, the relationship cache needs to be up-to-date.
See “Refreshing BaseTen's relationship cache” for details.

To list the properties:

Select the entity you want to inspect

Click the Inspector button on the Toolbar, or select Window → Get Info… ⌘I

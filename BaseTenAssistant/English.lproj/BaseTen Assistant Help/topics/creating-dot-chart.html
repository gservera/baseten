Creating a chart of the database schema

BaseTen Assistant can create a graphical representation of the database schema. The graph contains entities, attribute names and types and relationship names and types much like the one used in Xcode data modeler. The can be saved for viewing for either Graphviz or OmniGraffle.

For relationships to be listed correctly, the relationship cache needs to be up-to-date.
See “Refreshing BaseTen's relationship cache” for details.

To create a chart:

Select File → Export Object Model…

Choose the desired file format.

  • DOT for Graphviz creates a non-directional graph.
  • OmniGraffle can open DOT files but it doesn't represent all label types properly. Therefore there's a separate option for it. It doesn't necessarily work with astral Unicode characters, though.
  • There's also the option to create a file in intermediate XML format.
